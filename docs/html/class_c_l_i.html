Add an option.

This function inserts an option into the current group and saves the given information.

Parameters
shortoptThe short option to be used. Empty to have none.
longoptThe long option to be used. Empty to have none.
argA short description what the option's value is for. I.e.: <DIR>
descA description of the option.
optionalif true, then this option will be displayed as an optional parameter.
defaultValueThe value that should be present by default.
Note
You need either shortopt or longopt - or both. But you can not leave both empty.
Only if you supplied arg, you will be able to pull a value.
