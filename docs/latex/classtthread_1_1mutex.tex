\subsection{Detailed Description}
Mutex class. 

This is a mutual exclusion object for synchronizing access to shared memory areas for several threads. The mutex is non-\/recursive (i.\+e. a program may deadlock if the thread that owns a mutex object calls \hyperlink{classtthread_1_1mutex_a41add9fd340426bfe5a7eac19e326e2a}{lock()} on that object). \begin{DoxySeeAlso}{See also}
\hyperlink{classtthread_1_1recursive__mutex}{recursive\+\_\+mutex} 
\end{DoxySeeAlso}
